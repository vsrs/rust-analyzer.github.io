apply_demorgan

Source: apply_demorgan.rs

Apply [De Morgan’s law](https://en.wikipedia.org/wiki/De_Morgan%27s_laws). This transforms expressions of the form !l || !r into !(l && r). This also works with &&. This assist can only be applied with the cursor on either || or &&, with both operands being a negation of some kind. This means something of the form !x or x != y.

Before
1
2

    if x != 4 ||┃ !y {}
}
After
1
2

    if !(x == 4 && y) {}
}
